This is the README for Caleb Young's contribution to the project at the moment.

My current implementation is a pseudo-snake game to exploit the limitations set by deuteranopia.
The reason this is pseudo-snake is because the goal of this game is not to play snake. It is to help people become orinted with "having" deuteranopia.
The game works in similar practice: with a growing snake trying to eat fruit. I kept the UI on the mobile side, although if this gets moved to the hmd it will become more involved with a controller.

The key difference in this are the colors. Under normal color, the floor and fruit have very different colors. Under the deuteranopia filter, they appear almost identical. This is how this project will contribute to the final project.
I developed this on an Andriod, however the game is saved as a prefab, so it can be migrated relatively easily.
